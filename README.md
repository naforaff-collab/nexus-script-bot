# 𝐍𝐞𝐱𝐮𝐬 𝐒𝐜𝐫𝐢𝐩𝐭 — بوت التذاكر (النسخة المُصلحة)

## ⚙️ إعداد البوت

### 1. ملف `.env`
افتح ملف `.env` وضع:
```
BOT_TOKEN=توكن_البوت_هنا
GITHUB_TOKEN=توكن_جيتهاب_هنا (اختياري)
CLIENT_ID=معرف_البوت_هنا
```

### 2. تثبيت المكتبات
```bash
npm install
```

### 3. تسجيل الأوامر
```bash
npm run deploy
```

### 4. تشغيل البوت
```bash
npm start
```

## 🔧 ما تم إصلاحه

### ✅ أزرار التذكرة
- **إغلاق التذكرة** — يغلق ويوقف إرسال رسائل
- **استلام التذكرة** — يعين المسؤول ويمنع الآخرين
- **حذف التذكرة** — يحذف القناة بعد 5 ثوانٍ
- **إضافة عضو** — يفتح نموذج لإدخال ID العضو
- **إزالة عضو** — يفتح نموذج لإدخال ID العضو

### ✅ نظام التقييم (Ephemeral)
- يظهر فقط لصاحب التذكرة بعد الإغلاق
- **لا يظهر** داخل قناة التذكرة
- يُرسل إلى روم التقييم فقط

### ✅ نظام Transcript
- عند إغلاق التذكرة يُنشأ ملف .txt بجميع الرسائل
- يُرسل تلقائياً إلى روم اللوغات

### ✅ اللوغ الكامل
- لوغ فتح التذكرة
- لوغ إغلاق التذكرة + Transcript
- لوغ حذف التذكرة

### ✅ الأقسام الصحيحة
- طلب سكريبت | Script Request
- شراء رتبة | Purchase Rank
- الدعم الفني | Technical Support
- إعلان / تعاون | Advertisement
- شراء بوت | Purchase Bot
- شراء تصميم | Purchase Design
- التقديم للإدارة | Apply

### ✅ لوحة التحكم (control-panel)
- إضافة/تعديل/حذف الأقسام
- تحديد روم اللوغات
- تحديد روم التقييم
- تحديد الكاتاغوري
- تحديد رولات الستاف
- إرسال الواجهة

### ✅ لوحة تحكم التذاكر (ticket-panel)
- إحصائيات التذاكر
- تحديث واجهة التذاكر
- إغلاق جميع التذاكر

### ✅ الأمان
- التوكن في ملف `.env` فقط
- لا يوجد توكن داخل الكود
